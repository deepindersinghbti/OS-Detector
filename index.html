How to use the app

NOTE: You must be on a computer/laptop to use this app.

  1. Right-click anywhere on this web page.
  2. Click Inspect.
  3. You will find 4 tabs on the top: Elements, Console, Sources and Network. Click the Console tab.
  4. Now, you might see a few error messages in the console. Simply ignore them and try to find a message in the console in this format: "You are using __________ operating system."*
*(The blank represents variation in operating systems. You might be using Windows on your current computer, and MacOS on another. This is why no specific operating system has been mentioned.)
